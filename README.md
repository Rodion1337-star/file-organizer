# File Organizer

**Desktop-приложение для автоматической сортировки файлов**

Утилита, которая помогает быстро навести порядок в любой папке.

## Возможности

- Сортировка по типам (Изображения, Документы, Видео, Аудио, Архивы, Другое)
- Сортировка по дате (папки по годам и месяцам)
- Dry Run режим (показывает, что будет сделано)
- Прогресс-бар и подробный лог
- Настраиваемые расширения файлов
- Тёмная тема

## Технологии
- Python + CustomTkinter
